# Tetris-io 🎮

Современный, быстрый и функциональный клон Тетриса, написанный с нуля на Python с использованием библиотеки Pygame. Проект включает плавную механику, несколько игровых режимов, визуальные анимации очистки линий, звуковые эффекты и встроенное меню настроек.

## ✨ Особенности

- **Идеальное выравнивание:** Чистый и аккуратный интерфейс с использованием сглаженного шрифта `Segoe UI`.
- **Два игровых режима:**
  1. **Classic Mode:** Стандартный темп игры с постоянной скоростью падения фигур.
  2. **Speed Up Mode:** Динамичный аркадный режим, где скорость увеличивается на 10% за каждые 50 набранных очков.
- **Буфер удержания фигуры (Hold):** Возможность сохранить текущую фигуру или поменять её на ранее отложенную нажатием клавиши `C`.
- **Аудиосинтезатор:** Генерация фоновой музыки и звуковых эффектов в реальном времени — внешние аудиофайлы не требуются!
- **Меню настроек:** Удобное включение и выключение музыки или звуков (SFX) прямо во время работы приложения.
- **Визуальный отклик:** Аккуратная сетка строго внутри игрового стакана и яркие вспышки при сгорании рядов.

## 🕹️ Управление

Для максимального удобства можно использовать как клавиши **WASD**, так и **стрелочки**:

- **Движение влево/вправо:** `A`/`D` или `Стрелка влево`/`Стрелка вправо`
- **Мягкий сброс (ускорение):** `S` или `Стрелка вниз`
- **Поворот фигуры:** `W` or `Стрелка вверх`
- **Удержать/поменять фигуру:** `C`
- **Навигация в меню:** `W`/`S` (или стрелочки) для выбора, `Enter` или `Пробел` для подтверждения.

## 🚀 Как запустить

### Требования
Убедитесь, что у вас установлены Python 3.x и библиотека `pygame`:
```bash
pip install pygame
```

### Клонирование и запуск
```bash
git clone https://github.com/loanelly/Tetris-io
cd Tetris-io
python Tetris-io.py
```
