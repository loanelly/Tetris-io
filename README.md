# Tetris-io: Simple ML Edition 🤖🧠

Это оптимизированная, облегчённая и высокопроизводительная версия проекта **Tetris-io**, полностью очищенная от лишней мультимедийной нагрузки (без музыки, главного меню, искусственных задержек и экранов окончания игры). Она разработана специально для использования в качестве быстрой симуляционной среды для алгоритмов **машинного обучения**, агентов обучения с подкреплением (Reinforcement Learning) и ИИ-экспериментов.

## ✨ Ключевые ML-оптимизации

- **Мгновенный перезапуск:** При достижении условий проигрыша игровое поле автоматически очищается (`locked.clear()`), сбрасывает счёт и спавнит новые фигуры за доли миллисекунды. Никаких зависаний экрана или пауз.
- **Нулевая задержка:** Код полностью очищен от вызовов `pygame.time.delay`, `pygame.mixer`, аудиосинтезаторов и тяжёлых анимаций вспышек при очистке линий, которые снижали показатель FPS.
- **Стабильная среда:** Симуляция работает на фиксированной базовой скорости (`base_spd = 0.27`), что помогает модели выстраивать чёткие и предсказуемые зависимости для системы наград и штрафов.
- **Интерфейс механики Hold:** Сохранена работающая функция удержания фигуры `Hold (C)`. Состояние буфера автоматически сбрасывается в `None` при перезапуске игрового цикла.



## 🕹️ Управление (для ручного тестирования и отладки)

События клавиатуры напрямую привязаны к движку Pygame, что позволяет легко управлять процессом вручную или имитировать нажатия через скрипты:

- **Движение влево:** `A` или `Стрелка влево`
- **Движение вправо:** `D` или `Стрелка вправо`
- **Мягкий сброс (ускорение):** `S` или `Стрелка вниз`
- **Поворот фигуры:** `W` или `Стрелка вверх`
- **Удержать / поменять фигуру:** `C`



## 🚀 Как запустить

### Требования
Убедитесь, что у вас установлены Python 3.x и библиотека `pygame`:
```bash
pip install pygame
```

### Запуск симуляции
```bash
python Tetris-io-simple-version.py
```

## 📊 Интеграция с RL-фреймворками (например, Gymnasium)

Если вы планируете обернуть этот код в стандартную среду Gymnasium, все необходимые компоненты уже готовы:
- **Пространство состояний (Observation space):** Данные можно извлекать напрямую из словаря заблокированных позиций `locked` (сетка 10x22, где занятые ячейки возвращают цветовые кортежи).
- **Функция шага (Step function):** Срабатывает каждый раз при падении фигуры вниз или при обработке управляющего действия из цикла событий.
- **Сигналы наград (Reward signals):** Могут быть привязаны к выводу функции `clear_rows(locked)` (например, выдача положительного вознаграждения за каждую очищенную линию).
